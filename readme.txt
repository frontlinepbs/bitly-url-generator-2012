Automatically creates a [bit.ly](http://bit.ly) url for each of your posts when they get published. To display the shortened url, simple use [wp_get_shortlink](http://codex.wordpress.org/Function_Reference/wp_get_shortlink) in your template file.

== Installation ==

1. Upload the plugin to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a [bit.ly API key](http://bitly.com/a/your_api_key)
4. Enter your bit.ly login and API key on the Settings > bit.ly screen
5. Enjoy
